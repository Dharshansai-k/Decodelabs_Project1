# 🤖 Rule-Based AI Chatbot

## Project Overview

This project is developed as part of the Decode Labs Artificial Intelligence Internship Program.

The chatbot uses rule-based logic and predefined responses to simulate human interaction.

---

## Features

- Greeting responses
- Exit commands
- Dictionary-based response system
- Continuous interaction loop
- Input sanitization
- Fallback response handling

---

## Technologies Used

- Python 3
